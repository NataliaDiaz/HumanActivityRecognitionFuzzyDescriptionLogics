Fuzzy OWL2 package
-----------------------

1. The FuzzyOWL2 folder contains the standalone FuzzyOWL2 parser to convert FuzzyOWL2 ontologies into DeLorean and FuzzyDL syntax

2. The FuzzyOWL2PlugIn folder contains the plugin for the Protege editor (http://protege.stanford.edu/), to build Fuzzy OWL2 ontologies, query them via fuzzyDL, and converting into DeLorean and FuzzyDL syntax 

3. The documentation of both are in their respective folders.
